Sutter Health Occupancy Data Analysis Dashboard__Herrick

Data Science

Description

This is an occupancy data analysis dashboard for the Sutter Health Herrick Campus.

Dashboard (Part1) presents a comprehensive analysis of gross area (in square feet, SF) for the Sutter Health Herrick Campus by floor level and cost center, with filters available for department descriptions. Dashboard (Part2) provides a detailed breakdown of the comprehensive gross area (in square feet, SF) by department description and cost center code for each floor level.

Cost Center Code:

  • Each stacked bar segment represents a Cost Center, identified by a numeric code (e.g., 27433, 28756, 36172, etc.).

  • These cost centers are color-coded and correspond to specific departments within the hospital.

  • The color legend matches cost centers to departments, helping identify which services occupy which parts of the building.

Area (SF):

  • The Y-axis measures the total area in square feet (SF).

  • Each bar segment is labeled with the area occupied by that cost center on a specific floor level.

  • The total area per level is summed up and shown at the bottom left of Dashboard (Part1).

Level (Floor):

  • The X-axis represents different building levels

  • Each level has its own stacked bar, with heights proportional to the total gross area per floor.

  • The left sidebar highlights the selected level and its corresponding total area.

Filters – Department Description:

  • On the right side, there is a filter pane listing all available departments.

  • The dashboard currently shows data with all 58 departments selected.

  • Users can toggle these checkboxes to view area distribution for specific departments/services.
